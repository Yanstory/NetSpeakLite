# NetSpeakLite

基于本地 .NET 原生引擎的文字转语音（TTS）朗读器

---

## 起因

- 好友开黑直播
- 家里有人不便开麦
- 转了一圈全是大模型/AI/付费生成

## 如何使用

1. 编译
2. 启动NetSpeakLite
3. 将系统“录制设备”的“立体声混音”启用
4. 在对应软件中将麦克风设置为“立体声混音”
5. Enjoy

或者——

- 如果软件支持分流声音，你可以把这个程序挂接到那个声音输出源上

## 缺点？
- 立体声混音会将设备的所有音频（包括播放器等）转录到输出，因此如果这么做，你将不能播放音乐/后台开启声音挂游戏等
- 这是一个练手项目，不建议登大雅之堂（）
